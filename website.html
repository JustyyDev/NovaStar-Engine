NovaStar is a free game engine for people who want to actually finish a game instead of spending 6 months picking one. Visual editor, 3D modeler, custom scripting. No PhD required.

Why NovaStar?

Other engines made you read a 400-page manual.
We made ours fun.

Look, we get it. Most game engines feel like they were designed by robots for robots. NovaStar was built for actual humans who want to make actual games.

Visual Editor

Drag stuff around. Click buttons. See your game take shape in real time. It's like playing with LEGOs but on a computer.

3D Modeler

Make low-poly characters, houses, swords, whatever. Right inside the editor. No Blender degree needed.

Tilemap Editor

Paint levels tile by tile. Grass, stone, water, lava. Collision is generated automatically.

Audio

Built-in sound effects that just work. Or import your own WAV/MP3/OGG if you're fancy like that.

UI System

Health bars, score counters, menus, the whole deal. Because what's a game without UI?

NovaScript

Our scripting language reads like normal English-ish code. If you can write "if player hits enemy, ouch" you're halfway there.

Getting Started

From zero to your first game in about 5 minutes.

1

Install Node.js

Go to nodejs.org and download version 18 or newer.

2

Download NovaStar

Grab from GitHub Releases. Unzip it. Run npm install.

3

Start Building

Run npm run dev and open localhost:3000/editor.html. Pick a template. Hit Play.

What do I do next?

I'm a total beginner

Start with the 3D Platformer template. Poke around. Break things. The built-in tutorial at tutorial.html walks you through everything.

I know some code

Check the NovaScript console. Try engine.audio.play('jump'). The API docs cover every system.

I want to make 3D models

Open the 3D Modeler tab. Click "+ New Model" or try a preset. Save as Prefab and spawn it anywhere.

I want to join the game jam

Head to Events, register your team, and learn the engine before April 14.

NovaStarEngine

The main engine class. Handles the Three.js scene, renderer, camera, and game loop.

engine.init(container, options?)
Initialize the engine and attach it to a DOM container.
container HTMLElement - The DOM element to render into
options.antialias boolean - Enable antialiasing (default: true)
const engine = new NovaStarEngine();
engine.init(document.getElementById('game'), { antialias: true });
engine.addObject(name, mesh, options?)
Add a named object to the scene.
name string - Unique identifier
options.position [x,y,z] - Initial position
options.components string[] - Components to attach
Returns: GameObject
engine.getObject(name)
Retrieve a named object. Returns null if not found.
Returns: GameObject | null
engine.removeObject(name)
Remove an object and dispose its geometry/materials.

MeshBuilder

Create low-poly 3D primitives and combine them into prefabs.

MeshBuilder.box(w, h, d, color)
Create a box mesh with MeshToonMaterial.
const crate = MeshBuilder.box(1, 1, 1, '#8b7355');
engine.addObject('crate', crate, { position: [0, 0.5, 0] });
MeshBuilder.sphere(radius, color, segments?)
Create a sphere. Low segments = low-poly look.
MeshBuilder.cylinder(rTop, rBottom, height, color)
Create a cylinder or cone (rTop=0 for cone).
MeshBuilder.combine(meshes, name)
Merge meshes into a single group.

AudioEngine

Play sounds, music, and 3D positional audio. WAV/MP3/OGG.

engine.audio.play(name, options?)
Play a sound. Built-ins: jump, coin, hit, powerup, click, whoosh.
options.volume number 0-1
options.loop boolean
options.category 'sfx'|'music'|'voice'
engine.audio.play('coin', { volume: 0.8 });
engine.audio.play('bgm.mp3', { loop: true, category: 'music' });
engine.audio.playAt(name, position, options?)
Play 3D positional audio at a world position.

UISystem

In-game UI: health bars, score displays, menus, HUD overlays.

engine.ui.addBar(name, options)
Add a progress/health bar.
engine.ui.addBar('health', { max: 100, color: '#ff4757', position: 'top-left' });
engine.ui.setBar('health', 75);
engine.ui.addText(name, text, options)
Add a text label to the HUD.

Tilemap

2D tile-based levels with auto collision.

new Tilemap(width, height, tileSize)
Create a tilemap. Each tile = 3D box.
const map = new Tilemap(20, 15, 1);
map.setTile(5, 3, 'grass');
map.buildColliders();

ComponentSystem

Attach reusable behaviors. Components run update() every frame.

engine.addComponent(obj, name, config?)
Built-in: PlayerController, PatrolAI, Collider, RigidBody, Spawner, HealthSystem, Collectible.
engine.addComponent('player', 'PlayerController', { speed: 5, jumpForce: 8 });
engine.addComponent('enemy', 'PatrolAI', { speed: 2, range: 5 });

PrefabSystem

Save and instantiate reusable game object templates.

PrefabLibrary.register(name, config)
Register a prefab template.
PrefabLibrary.register('coin', {
  mesh: MeshBuilder.cylinder(0.3, 0.3, 0.1, '#ffd84d'),
  components: ['Collectible', 'Spin'],
  properties: { value: 10 }
});
PrefabLibrary.spawn('coin', { position: [3, 1, 0] });

NovaScript

Custom scripting language for game logic. .nova files.

Script Structure
Define on_start, on_update, and on_event handlers.
on_start {
  set health = 100
  set speed = 5
}

on_update {
  if key_pressed("space") {
    jump(8)
    play_sound("jump")
  }
}

on_event "hit" {
  set health = health - 10
  if health <= 0 { game_over() }
}
Built-in Functions
Helper functions that map to engine systems.
move(x,y,z) jump(force) teleport(x,y,z)
play_sound(name) stop_sound(name)
spawn(prefab,x,y,z) destroy(name)
wait(seconds) random(min,max) distance(a,b)
set_bar(name,val) show_text(name,text)
