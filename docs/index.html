Why NovaStar?

Other engines made you read a 400-page manual.
We made ours fun.

Look, we get it. Most game engines feel like they were designed by robots for robots. NovaStar was built for actual humans who want to make actual games.

Visual Editor

Drag stuff around. Click buttons. See your game take shape in real time. It's like playing with LEGOs but on a computer. (And nobody steps on them.)

3D Modeler

Make low-poly characters, houses, swords, whatever. Right inside the editor. No Blender degree needed. Your models become prefabs you can reuse everywhere.

Tilemap Editor

Paint levels tile by tile. Grass, stone, water, lava (for when you want to be mean to your players). Collision is generated automatically.

Audio

Built-in sound effects that just work. Jump? Boing. Collect a coin? Sparkly chime. Or import your own WAV/MP3/OGG if you're fancy like that.

UI System

Health bars, score counters, menus, the whole deal. Because what's a game without a way to tell the player they're losing?

NovaScript

Our scripting language reads like normal English-ish code. If you can write "if player hits enemy, ouch" you're basically halfway there already.

Getting Started

From zero to your first game in about 5 minutes. Not joking. We timed it.

1

Install Node.js

Go to nodejs.org and download version 18 or newer. It's free, takes 2 minutes, and you only have to do it once.

2

Download NovaStar

Grab the latest release from GitHub Releases. Unzip it. Open a terminal in that folder and run npm install.

3

Start Building

Run npm run dev and open localhost:3000/editor.html. Pick a template. Add stuff. Hit Play. You just made a game. (Sort of.)

What do I do next?

I'm a total beginner

Start with the 3D Platformer template. It comes with a player, ground, stars to collect, and an enemy. Poke around. Move things. Break things. That's how you learn. The built-in tutorial at tutorial.html walks you through everything with pictures and code examples.

I know some code

Check out the NovaScript console in the editor (bottom panel, Console tab). Try typing engine.audio.play('jump') and hit enter. Then look at the Star Hopper demo game in src/game/ to see how a real game is structured. The API docs in tutorial.html cover every system.

I want to make 3D models

Open the editor, go to the 3D Modeler tab at the bottom. Click "+ New Model" or try a preset (House, Tree, Sword). Add primitives, set colors, position them. When it looks right, hit "Save as Prefab" and you can spawn it anywhere in your game.

I want to join the game jam

Awesome. Head to the Events page, scroll to the Spring 2026 Game Dev Race, and register your team. Then spend the signup period (now through April 13) learning the engine. When the theme drops April 14, you'll be ready.
